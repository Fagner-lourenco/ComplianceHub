# Blueprint da V2 do ComplianceHub

## Visao de produto

ComplianceHub V2 deve ser uma plataforma investigativa corporativa BigDataCorp-first para due diligence, background check, analise de risco, investigacao de PF/PJ, vinculos, evidencias, revisao humana e relatorios executivos auditaveis.

## Posicionamento

Produto para empresas que precisam transformar dados dispersos em decisoes defensaveis:

- Compliance.
- Juridico.
- RH.
- Operacoes.
- Franquias.
- Credito.
- Onboarding.
- Gestao de risco reputacional.
- Due diligence de fornecedores, socios, candidatos e clientes.

## Core da V2

Entram obrigatoriamente:

- BigDataCorp-first provider layer.
- Raw snapshots versionados.
- Entidades canonicas PF/PJ.
- Evidence store.
- Dossie investigativo.
- Case workflow com revisao humana.
- Risk signals.
- Decision explicavel.
- Relatorio versionado.
- Projecao segura para cliente.
- Auditoria operacional e de evidencia.

## Opcional na V2

- Grafo visual simples.
- Templates de relatorio por segmento.
- Modulos por plano.
- Watchlist manual basica.
- Comparacao entre consultas anteriores.

## Fica para V3

- Monitoramento continuo completo.
- Rule builder no-code.
- SDK/API publica ampla.
- BI avancado.
- Marketplace de providers.
- Grafo investigativo avancado.
- Automacoes complexas por workflow builder.

## Diferencial

O diferencial da V2 nao e consultar dados. E provar, organizar, explicar e entregar uma decisao revisada.

```txt
Consulta -> Evidencia -> Fato -> Relacao -> Sinal -> Decisao -> Relatorio
```
