https://plataforma.bigdatacorp.com.br/uso/

The Usage Statistics API returns information about service usage (and the limits configured for this usage) by the user performing the query, within the period specified by the user. Queries are counted dynamically, so a request made now and another made five minutes from now could potentially bring different results. Verification in very long periods (date ranges ) can impact the response time of the request.

This API returns information about the total number of requests made against the service, the number of requests that returned successfully and in error, the average response time for the requests, and an estimate of the amount that will be charged to the customer for the requests made ( this value may change due to the volume of queries performed ). In addition to the general numbers, detailed statistics are also sent for each Dataset available to the customer, allowing a detailed analysis of the service's behavior.

Finally, the API return also contains the monthly query limit configured for the user's account, and allows a follow-up of whether the query volume has already reached this limit or not.

Operation

If a parameter is not passed in the Body, all calls made by it in the current month will be counted. Alternatively, calls to the Usage Statistics API can be assigned a start date (InitialReferenceDate) and an end date (FinalReferenceDate), which will be used to compose the period in which the requests made will be counted. In addition, it can also receive a date format parameter (DateFormat), which indicates in which format the dates given in the input were filled in.

When we are composing the period, the 0:00 hours of each informed date are considered, that is, a request that receives January 1st as the starting date and December 31st as the end date will only count the requests made at 0:00 pm on the day January 1st until 12:00 am on December 31st, and calls made during the 31st will be ignored. To account for an entire year, the 1st of January of the year you wish to consult must be passed as the start date, and the 1st of January of the following year as the end date.

Examples

How to retrieve usage statistics for the current month:

POST https://plataforma.bigdatacorp.com.br/uso
Body:
{}


How to retrieve usage statistics for the month of July 2019:

POST https://plataforma.bigdatacorp.com.br/uso
Body:
{
  "InitialReferenceDate": "2019-07-01",
  "FinalReferenceDate": "2019-08-01",
  "DateFormat": "yyyy-MM-dd"
}


How to retrieve usage statistics for the year 2018:

POST https://plataforma.bigdatacorp.com.br/uso
Body:
{
  "InitialReferenceDate": "2018-01-01",
  "FinalReferenceDate": "2019-01-01",
  "DateFormat": "yyyy-MM-dd"
}
