In this section, you'll find everything you need to start using BigDataCorp's APIs safely and efficiently:

Prerequisites: Know what is needed before starting: credentials, access to the BDC Center, and user permissions.
First consultation: learn how to generate your AccessToken and TokenId, add them correctly to the request headers, and see practical examples of real calls to test your integration.
Tips and best practices: learn about our recommendations for security, token rotation, and integration stability.

Explore the pages in the sidebar menu and, if you have any questions, speak to the team responsible for your support.
