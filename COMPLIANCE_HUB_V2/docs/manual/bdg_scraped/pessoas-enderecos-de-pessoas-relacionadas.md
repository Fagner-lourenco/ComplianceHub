Description

This dataset returns addresses associated with people related to the individual being consulted, allowing for the analysis of indirect links, address characteristics, recency, priority, and nature of the association. The data supports relationship analysis, segmentation, and automated decision-making in flows that require enhanced contact context.

For more information on business context, applications, and use cases, please refer to the Contacts documentation.

Technical name of the dataset: related_people_addresses.

This dataset allows you to limit the number of records returned using the modifier .limit(x) in the field Datasets.
