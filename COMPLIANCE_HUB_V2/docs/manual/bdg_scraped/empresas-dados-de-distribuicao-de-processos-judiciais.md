Description

This dataset consolidates aggregated information about legal proceedings in which the consulted entity is involved, allowing for a quick overview of the distribution of these proceedings across different dimensions, such as type, court, state, status, and party role. The return is based on the main fields of the Legal Proceedings dataset, making it suitable for exploratory analyses and preliminary assessments, without the need to consume large volumes of detailed data. Distribution values ​​are aggregated in real time, prioritizing the most recent data. In cases of entities with more than 500,000 proceedings, the return may be empty, accompanied by an error. UNABLE TO AGGREGATE DATA IN TIME (code -1215).

For more information on business context, applications, and use cases, please refer to the Legal Proceedings documentation.

Technical name of the dataset: lawsuits_distribution_data.
